Q.10] Explain hard and soft mount,tcp/udp.

Hard Mount

The user cannot terminate the process waiting for NFS communication to resume.

Soft mount

The user will get error alert when NFS server is not reachable.

TCP/UDP

NFS is client and server architecture based protocol and it can be used transport layer as both depend upon the application requirement.

To increase/decrease the data segment size of a process

To bias the existing priority of a process
