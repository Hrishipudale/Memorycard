Q.3] Explain the different types of owners in Linux file system

User

It is the owner of the file. By default, the person who created a file becomes its owner

Group

A user- group can contain multiple users. All users belonging to a group will have the same access permissions to the file.

Other

Any other user who has access to a file. This person has neither created the file, nor he belongs to a usergroup who could own the file

Allows everyone to read and write to the file

Allows only the group to read and execute (not write)
