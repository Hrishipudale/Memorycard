Q.6] What do these settings do when used with chmod: 1) a+rw 2) ug+r 3) g=rx

a+rw

Allows everyone to read and write to the file

ug+r

Allows the owner and group to read the file

g = rx

Allows only the group to read and execute (not write)

To increase/decrease the data segment size of a process

To bias the existing priority of a process
