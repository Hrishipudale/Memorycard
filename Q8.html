Q.8] Explain the different types of commands used in linux Administrative Task

free

This command shows the free and used memory (RAM) on the Linux system.

kill

This command is used to terminate running processes on Linux machine. To use these utilities you need to know PID of process

top

This command is used to list all the running processes on the Linux machine.
