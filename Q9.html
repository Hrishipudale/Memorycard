Q.9]How to list file permissions in octal format?

$ stat -c "%a %n" *

To get octal permissions of all the files and directories in the current working directory

$ stat -c "%a %n" /usr/share/

To get octal permissions only for the directory, say /usr/share, and not the content inside it

$ stat -c "%a %n" /usr/share/*
